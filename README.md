# Video Combiner Tool

A web-based tool for combining multiple videos with randomized cuts using FFmpeg.

## Features

- Upload multiple videos
- Configure min/max cut durations
- Randomized segment extraction
- Fast-cut video combination
- Download combined output

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the server:
```bash
npm start
```

3. Open browser to: http://localhost:8080

## Requirements

- Node.js
- FFmpeg
- Modern web browser

## Usage

1. Add videos by dragging or clicking
2. Configure cut duration settings
3. Click "Process Videos"
4. Download the combined result
